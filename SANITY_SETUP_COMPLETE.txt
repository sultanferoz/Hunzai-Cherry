================================================================================
                    THE 3 KEYS YOU NEED TO GET
================================================================================

KEY 1: PROJECT ID
       From: sanity.io > Your Project > Settings > API > "Project ID"
       Format: 20-character code (abc123def456ghijkl)
       Your ID: _________________________________

KEY 2: DATASET  
       From: sanity.io > Your Project > Settings > API > "Dataset"
       Standard: production
       Your Dataset: _________________________________

KEY 3: API TOKEN
       From: sanity.io > Your Project > Settings > API > Tokens > "Add API token"
       Format: sk_production_... (long string)
       Permissions: Must be set to "Editor"
       Your Token: _________________________________

================================================================================
                         5-MINUTE SETUP PROCESS
================================================================================

STEP 1: CREATE SANITY PROJECT (2 minutes)
   1. Go to https://www.sanity.io
   2. Create account / Log in
   3. Click "Create Project"
   4. Project name: Cherry
   5. Template: Blank
   6. Dataset: production
   7. Click "Create"
   → SAVE YOUR PROJECT ID

STEP 2: CREATE API TOKEN (1 minute)
   1. Go to Settings > API > Tokens
   2. Click "Add API token"
   3. Name: "Cherry Frontend"
   4. Permissions: "Editor"
   5. Click "Save"
   → SAVE YOUR API TOKEN

STEP 3: ADD TO YOUR APP (1 minute)
   1. Create file: .env.local
   2. Copy this template:
      NEXT_PUBLIC_SANITY_PROJECT_ID=your-project-id
      NEXT_PUBLIC_SANITY_DATASET=production
      SANITY_API_TOKEN=your-api-token
   3. Replace with your actual values
   4. Save file

STEP 4: START YOUR APP (1 minute)
   1. Run: pnpm install
   2. Run: pnpm dev
   3. Visit: http://localhost:3000
   4. ✅ Done! App is connected to Sanity

STEP 5: ADD YOUR FIRST PRODUCT
   1. Go to sanity.io dashboard
   2. Click "Products"
   3. Click "Create"
   4. Fill in: name, price, emoji, image, description
   5. Click "Publish"
   6. Refresh localhost:3000
   7. ✅ Product appears on your site!

================================================================================
                           TROUBLESHOOTING
================================================================================

ISSUE: "Cannot read properties of undefined"
FIX: Add missing NEXT_PUBLIC_SANITY_PROJECT_ID to .env.local
     Restart dev server after adding env vars

ISSUE: Content not appearing on site
FIX: Make sure content is "Published" (not draft) in Sanity
     Refresh your browser
     Wait 5 seconds for cache refresh

ISSUE: Images not loading
FIX: Check image uploaded in Sanity
     Set hotspot on image (image focus point)
     Clear browser cache (Ctrl+Shift+Del)
     Try different image format

ISSUE: API Token error
FIX: Verify token has "Editor" permissions in Sanity
     Token might be expired - generate new one
     Update .env.local with new token
     Restart dev server

ISSUE: Dev server won't start
FIX: Make sure Node.js 18+ is installed
     Run: rm -rf .next node_modules && pnpm install
     Restart: pnpm dev
